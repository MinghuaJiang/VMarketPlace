2. Our code use AWS Cognito, AWS Mobile Hub, AWS S3, AWS DynamoDB and AWS Analytics based on AWS Pinpoint.
awsconfiguration.json under src/main/res/raw is the aws configurartion file generated by AWS Mobile hub.

edu.virginia.cs.vmarketplace.service.client.AWSClientFactory is the factory class to build the client to access other AWS services leveraging the configuration above.

package edu.virginia.cs.vmarketplace.service.login and edu.virginia.cs.vmarketplace.view.login includes code dealing with user sign in based on AWS Cognito.

edu.virginia.cs.vmarketplace.service.S3Service is the service class we encapsulate to access AWS S3.

edu.virginia.cs.vmarketplace.service.AnalyticService is the service class we built to access AWS pinpoint to log and analyze user session.

package edu.virginia.cs.vmarketplace.service, edu.virginia.cs.vmarketplace.service.dao, edu.virginia.cs.vmarketplace.service.loader,
edu.virginia.cs.vmarketplace.model are the classes we used to model and access AWS dynamoDB.
